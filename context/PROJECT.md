# FINANHUB: Project Overview

- Marketplace de negócios e intermediação financeira de alta performance (Deal Flow).
- Plataforma premium para conexão entre investidores, empreendedores e ativos corporativos.

## Core Modules
- **Oportunidades**: Dashboard de exploração de ativos com filtros avançados.
- **Detalhe de Anúncio**: Análise profunda de ativos com métricas financeiras.
- **Leads**: Sistema de captura e gestão de intenções de negócio.
- **Dashboard Investidor**: Painel de acompanhamento de propostas e ativos favoritos.
- **Admin (Central de Lead)**: Controle operacional completo do pipeline de intermediação.

## Value Proposition
- Curadoria de ativos financeiros e corporativos.
- Intermediação assistida e fluxos de governança.
- Inteligência de dados (HAYAI AI) para qualificação de oportunidades.
