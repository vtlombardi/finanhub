# UI & UX Guidelines

- **Theme**: Dark Premium (#020617 base background).
- **Accents**: Teal / Aqua (#12b3af / #00b8b2).
- **Style**: Institutional / Investment-grade (Not Landing Page style).
- **Rule**: NO redesign of layouts unless explicitly requested.

## Layout & Components
- **Container**: Strict adherence to `max-w-7xl mx-auto`.
- **Hero**: Compact and functional header; must not dominate the viewport height.
- **Sidebar**: Defined as a functional tool; high density, strictly not decorative.
- **Spacing**: Rigid vertical rhythm; minimal padding; avoid excessive gaps.
- **Typography**: Clean hierarchy; avoid overuse of `font-black` or aggressive weights.

## Visual Control
- **Materiality**: Subtle borders (`border-white/5`); limited use of `glow`, `blur`, and `gradients`.
- **Intentionality**: Every element must serve a functional/analytical purpose.
- **Micro-interactions**: Subtle, fast transitions; no staggered entry animations.
